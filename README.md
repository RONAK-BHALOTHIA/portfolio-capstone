## Overview

This repository contains the source code for my personal portfolio site. The goal is to present cybersecurity work in a format recruiters and hiring managers can scan quickly: clear project highlights, relevant skills, earned certifications, and a concise professional narrative.

**Live demo:** Coming soon — deployment planned via GitHub Pages.

## Highlights

- **Cybersecurity-first portfolio** — Designed to feature security labs, projects, and hands-on experience
- **Clean, accessible web stack** — Semantic HTML, organized CSS, and vanilla JavaScript (no unnecessary frameworks)
- **Recruiter-friendly structure** — Sections built for fast scanning: who you are, what you've built, and how to contact you
- **AI-assisted development** — Built as part of the AI Fluency program, demonstrating modern development workflows

## Features

## Getting Started

No build step is required — this is a static site.

```bash
# Clone the repository
git clone https://github.com/RONAK-BHALOTHIA/portfolio-capstone.git
cd portfolio-capstone

# Preview locally (once index.html is added)
npx serve .
```

Alternatively, open `index.html` directly in your browser after the site structure is in place.
